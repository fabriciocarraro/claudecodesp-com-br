# Builders SP

Landing page oficial do Builders SP, feita com React + Vite.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Deploy

- O deploy esta configurado em `.github/workflows/deploy.yml`
- O `base` do Vite esta em `/claudecodesp-com-br/` para GitHub Pages.
